# 🚀 餐廳分析器 - 快速入門指南

## 📦 下載與安裝
### 🍎 **Mac 用戶**

1. **前往** [GitHub Releases](https://github.com/ntoujavaproject/store-strategist/releases) 頁面
2. **下載** 最新版本的 `RestaurantAnalyzer-v1.0.3.dmg`
3. **雙擊** 掛載磁碟映像
4. **拖拽** 應用程式到 Applications 資料夾
5. **啟動** 從 Applications 或 Launchpad

**⚠️ 系統需求：**
- macOS 10.14 或更新版本
- 需預先安裝 Java 21+（[點此下載](https://adoptium.net/)）

## 🎯 首次設定

### AI 初始化（Mac）
首次啟動應用程式時：

1. **AI 狀態檢查** - 應用程式檢查 Ollama AI 安裝狀態
2. **模型下載** - 下載 Gemma 3:4B 模型（約 2.6GB）
3. **初始化** - 設定 AI 服務（5-10 分鐘）
4. **完成！** - 應用程式準備就緒

**💡 小提示：**
- 確保網路連線穩定
- 初始化期間請勿關閉應用程式
- 下載僅需進行一次
- AI 分析使用本地精選評論，提高效能

## 🔍 使用應用程式

### 1. **餐廳搜尋**
- 在搜尋欄輸入餐廳名稱
- 從建議中選擇或繼續輸入
- 從結果中確認正確的餐廳
- 等待數據收集完成

### 2. **AI 聊天助手**
- 點擊右上角的聊天按鈕
- 詢問餐廳經營相關問題
- 獲取市場趨勢洞察
- 接收個人化建議

### 3. **數據分析**
- 查看評分分佈和趨勢
- 分析顾客評論
- 檢查價格和競爭情況
- 導出數據進行進一步分析

## ⚙️ 設定與偏好

透過齒輪圖示存取設定：

- **主題**：在淺色和深色模式間切換
- **字體**：調整文字大小和字體系列
- **AI**：配置 AI 聊天偏好設定
- **數據**：管理本地數據存儲

## 🛠️ 故障排除

### 常見問題和解決方案

#### ❌ **「應用程式無法啟動」**
**解決方案：**
- 確保檔案完整解壓縮
- 檢查防毒軟體設定
- 嘗試以管理員身分執行
- 驗證系統需求

#### ❌ **「AI 功能無法使用」**
**解決方案：**
- 檢查網路連線
- 等待首次 AI 模型下載完成
- 重新啟動應用程式
- 清除暫存資料

#### ❌ **「Java 錯誤訊息」**
**解決方案：**
- 確保已安裝 Java 21+
- 檢查 JAVA_HOME 環境變數
- 從 Adoptium 下載最新版 Java

#### ❌ **「AI 分析速度慢」**
**解決方案：**
- 首次使用需要下載 AI 模型
- 系統會優先使用精選評論加速分析
- 確保網路連線穩定
- v1.0.3 版本已大幅優化分析速度

#### ❌ **「數據收集失敗」**
**解決方案：**
- 檢查網路連線
- 確認餐廳名稱正確
- 重試搜尋
- 嘗試不同的餐廳名稱變體

# 🚀 餐廳分析器 - 快速入門指南

## 📦 下載與安裝

### 🪟 **Windows 用戶**

#### 選項 1：零安裝完整包（推薦）⭐
**適合沒有安裝 Java 或偏好無腦安裝的用戶**

1. **下載** `RestaurantAnalyzer-ZeroInstall-v1.0.0.zip`（261MB）
2. **解壓縮** 到任意資料夾（桌面、文件等）
3. **雙擊** `RestaurantAnalyzer.bat`
4. **等待** 3-5 分鐘進行首次設定
5. **開始** 分析餐廳！

**✅ 優勢：**
- ✨ 無需安裝 Java
- 🔧 包含完整運行環境
- 💻 適用於任何 Windows 10/11 電腦
- 🚀 一次設定，永久使用

#### 選項 2：智能安裝腳本
**適合偏好最小下載的用戶**

1. **下載** `SmartInstaller.bat`（10KB）
2. **雙擊** 執行
3. **等待** 自動下載和配置
4. **應用程式** 自動啟動

**✅ 優勢：**
- 📁 檔案極小
- 🤖 智能環境檢測
- ⚡ 自動化安裝流程

### 🍎 **Mac 用戶**

1. **下載** `RestaurantAnalyzer-Working.dmg`（6.8MB）
2. **雙擊** 掛載磁碟映像
3. **拖拽** 應用程式到 Applications 資料夾
4. **啟動** 從 Applications 或 Launchpad

**⚠️ 系統需求：**
- macOS 10.14 或更新版本
- 需預先安裝 Java 21+（[點此下載](https://adoptium.net/)）

## 🎯 首次設定

### AI 初始化（Windows 和 Mac）
首次啟動應用程式時：

1. **AI 狀態檢查** - 應用程式檢查 Ollama AI 安裝狀態
2. **模型下載** - 下載 Gemma 3:4B 模型（約 2.6GB）
3. **初始化** - 設定 AI 服務（5-10 分鐘）
4. **完成！** - 應用程式準備就緒

**💡 小提示：**
- 確保網路連線穩定
- 初始化期間請勿關閉應用程式
- 下載僅需進行一次

## 🔍 使用應用程式

### 1. **餐廳搜尋**
- 在搜尋欄輸入餐廳名稱
- 從建議中選擇或繼續輸入
- 從結果中確認正確的餐廳
- 等待數據收集完成

### 2. **AI 聊天助手**
- 點擊右上角的聊天按鈕
- 詢問餐廳經營相關問題
- 獲取市場趨勢洞察
- 接收個人化建議

### 3. **多餐廳比較**
- 點擊「新增分頁」開啟新分析
- 搜尋不同餐廳
- 在不同分頁間並排比較
- 輕鬆切換分頁

### 4. **數據分析**
- 查看評分分佈和趨勢
- 分析顾客評論
- 檢查價格和競爭情況
- 導出數據進行進一步分析

## ⚙️ 設定與偏好

透過齒輪圖示存取設定：

- **主題**：在淺色和深色模式間切換
- **字體**：調整文字大小和字體系列
- **AI**：配置 AI 聊天偏好設定
- **數據**：管理本地數據存儲

## 🛠️ 故障排除

### 常見問題和解決方案

#### ❌ **「應用程式無法啟動」**
**解決方案：**
- 確保檔案完整解壓縮
- 檢查防毒軟體設定
- 嘗試以管理員身分執行
- 驗證系統需求

#### ❌ **「AI 功能無法使用」**
**解決方案：**
- 檢查網路連線
- 等待首次 AI 模型下載完成
- 重新啟動應用程式
- 清除暫存資料

#### ❌ **「Java 錯誤訊息」**
**解決方案：**
- 使用零安裝版本（包含 Java Runtime）
- 或手動安裝 Java 21+
- 檢查 JAVA_HOME 環境變數

#### ❌ **「Python 相關錯誤」**
**解決方案：**
- 核心功能仍正常運作
- 僅數據收集功能可能受限
- 可忽略這些錯誤繼續使用

#### ❌ **「數據收集失敗」**
**解決方案：**
- 檢查網路連線
- 確認餐廳名稱正確
- 重試搜尋
- 嘗試不同的餐廳名稱變體

# 🍔 餐廳分析器 - 快速開始指南

## 📦 只需要兩個檔案

1️⃣ **智能安裝.bat** (10KB) - 智能安裝腳本  
2️⃣ **餐廳分析器-智能安裝版.zip** (250MB) - 完整應用程式包

## 🚀 使用方法（超簡單）

### Windows 用戶：
1. 下載 `dist` 目錄中的兩個檔案
2. 雙擊 `智能安裝.bat`
3. 等待 3-5 分鐘自動配置
4. 程式會自動啟動

### Mac 用戶：
1. 下載 `RestaurantAnalyzer-Working.dmg`
2. 雙擊安裝並拖拽到應用程式資料夾
3. 啟動餐廳分析器

## ✅ 智能特色

- **🔍 自動檢測**：檢查系統 Java 版本
- **📦 智能選擇**：
  - 有 Java 21+ → 使用系統 Java（節省空間）
  - 無 Java → 自動下載配置內建 Java
- **🎛️ 完全自動化**：無需用戶選擇或配置

## 📊 系統需求

### Windows
- Windows 10/11
- 至少 4GB RAM
- 500MB 硬碟空間
- 網路連線（首次啟動）

### Mac
- macOS 10.14+
- 至少 4GB RAM
- 200MB 硬碟空間
